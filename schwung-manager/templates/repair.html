Schwung Repair

{{if .NeedsRepair}}

Schwung's framework components are out of date and need repair.

The web manager can't complete this itself — it needs a one-time action with elevated access. Pick whichever path is easier:

Option 1 — SSH (one command, fastest)

From a terminal on the same network — macOS/Linux, Windows PowerShell, or WSL. Replace {{.Hostname}} with your device's IP if mDNS isn't available. On Windows, the GUI installer below is the easiest option. 

ssh root@{{.Hostname}} 'chown root:root /data/UserData/schwung/bin/schwung-heal && chmod 4755 /data/UserData/schwung/bin/schwung-heal && /data/UserData/schwung/bin/schwung-heal --reboot'

Move will reboot once. After it comes back up, this banner will be gone.

Option 2 — GUI Installer (no terminal needed)

Download the latest installer from the installer releases page, run it, and click Reinstall. The installer SSHes as root and sets up the missing pieces.
