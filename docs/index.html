Modules

Instruments & effects for the Ableton Move
8 modules 4 families free & open source MIT licensed interactive manuals

Loop glitching, live vocal processing, performance looping, and a six-track machine synth — all running natively on the Move via Schwung, the framework for custom modules on Ableton Move hardware. Every module ships with a full interactive operation manual and on-device help.

👊

Smack

Seeded loop glitcher · three builds, one core

Grab a quantized loop of live audio, auto-slice it, and let seeded randomness assign glitch effects and a new play order to the slices. Inspired by Sugar Bytes Looperator and dblue Glitch — patterns repeat identically every pass until you re-roll, and the Seed knob browses them by number.

📖 Operation manual Source

Smack

Audio FXv0.13.3

Runs in Signal Chain slots and Master FX — glitch a synth, or the whole Move mix.

  • Retroactive capture: loop the last 1 step – 16 bars you just played
  • 26 effects: retrigger, varispeed, tape stop, vinyl scratch, vowel filter, granular scatter…
  • Seeded per-slice FX + slice-reorder patterns; step buttons edit & mute
  • A/B punch between clean loop and pattern; dual-mono lanes with per-side patterns
Module Store: smack · Custom install: timncox/schwung-smack-fx

Smack In

Voicev0.13.3

Standalone build that reads Move's selected input — mic, line-in, or USB-C — directly in one slot. No routing required.

  • Same capture / slice / seed engine as Smack
  • Perfect for glitching external gear through the Move
Module Store: smack-in · Custom install: timncox/schwung-smack-voice

Oversmack

Overtakev0.10.2

Full-surface Smack: the whole pad grid becomes a step-FX editor for the input loop.

  • Steps show the pattern and select slices; playhead chase
  • Upper pad rows are a rearrangeable 22-effect palette — pins survive Re-Roll
  • Move's clock keeps running; Back hides the UI while audio keeps processing
Module Store: oversmack · Custom install: timncox/schwung-oversmack

Belt

Live vocal processor · named because it makes you sound like you can

Autotune-style pitch correction, up to four diatonic backing-vocal harmonies, a stereo doubler, and a formant knob — on the Move's own mic, line-in, or USB-C input. One YIN pitch analysis drives every TD-PSOLA voice, so seven voices cost barely more than one, at ~26 ms fixed latency.

📖 Operation manual Source

Belt

Audio FXv0.1.2

Chain + Master FX build — processes whatever is upstream in the signal chain.

  • Retune 0 = instant hard-tune robot, higher = transparent correction
  • Harmony voices 1–4: scale-aware 3rds, 5ths, octaves with per-voice detune & spread
  • HARD pad for momentary or latched T-Pain punch
  • Tuner strip on steps 1–12; Flex / Humanize keeps your slides and vibrato
Module Store: belt · Custom install: timncox/schwung-belt-fx

Belt In

Voicev0.1.2

The live-vocals build: a standalone slot synth on the hardware input, same chain UI and engine.

  • Sing straight into the Move's mic, line-in, or USB-C
  • Feedback-safe: auto-mutes when the speaker would howl (Monitor pad overrides)
Module Store: belt-in · Custom install: timncox/schwung-belt-voice

Mark

Five-track live loop station · named after Marc Rebillet

The Boss RC-505 workflow on the Move's pad grid: five independent stereo loop tracks cycling record → play → overdub, measure-snapped to the first loop or Move's MIDI clock. Move keeps playing underneath, so its own tracks are your rhythm section.

📖 Operation manual Source

Mark

Overtakev0.4.0

Per-track stop/clear, level, pan, reverse and one-shot, with single-level undo/redo of the last take.

  • 16 session slots: loop audio + settings, saved on the Move, survive reinstalls
  • Per-track FX insert: built-ins or any chainable Schwung Audio FX (yes, including Smack and Belt)
  • Single mode turns the five tracks into exclusive song sections
  • Web mixer at move.local: faders, pans, FX, modes, session grid
Module Store: mark · Custom install: timncox/schwung-mark
🎛️

Mono

Clean-room, Monomachine-inspired digital instrument

Six machine types — SuperWave Saw / Pulse / Ensemble, a SID 6581-style oscillator, a DigiPRO-style 12-bit wavetable, and two-block FM — each with an AHDR envelope, dual-cutoff resonant filter, distortion, EQ, sample-rate reduction, stereo filtered delay, and three assignable LFOs with 114 destinations.

📖 Operation manual Source

Mono

Overtakev0.3.1

Full-surface six-track machine synth with an internal 64-step sequencer.

  • Per-step parameter locks, probability, retrigs, conditions, and slide
  • Independent track timing, arp, neighbor routing, and song mode
  • Per-machine sound memory: switch machines and your settings come back
Custom install: timncox/schwung-mono

Mono Voice

Voicev0.3.1

One machine voice inside a normal Move track, driven by Move's own pads and sequencer.

  • Same six machines and full synthesis panel as Mono
  • Shift-banks add Drift, Fold, Bits, Noise, envelope curves, filter drive & more
Custom install: timncox/schwung-mono-voice

Get started

1

Install Schwung

All modules run on Schwung, the open framework for custom JavaScript and native DSP modules on the Ableton Move.

2

Open the manager

Browse to move.local:7700 on the same network as your Move and open the Module Store.

3

Install a module

Search the store by name, or use Install Custom Module with any of the repository links above. Updates show up in the manager automatically.
